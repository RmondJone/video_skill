# 视频画面解说生成器技能设计

**日期：** 2026-03-19
**技能名称：** video-recognition
**状态：** 已批准

---

## 概述

**视频画面解说生成器** - 一个独立的新技能，专注于无声视频的画面理解与解说文案生成。

**核心定位：** 不依赖语音识别，完全基于画面分析生成风格化解说文案。

---

## 风格选项（5种）

| 风格 | 说明 | 适用场景 |
|------|------|----------|
| 风趣幽默（默认）| 轻松调侃的语气，画外音风格 | 日常vlog、娱乐内容 |
| 技术硬核 | 专业术语、深度分析 | 科技、编程、工业 |
| 理性科普 | 客观陈述、逻辑清晰 | 知识讲解、教育 |
| 解压治愈 | 柔和舒缓、放松心情 | ASMR、冥想，自然 |
| 温馨感人 | 情感充沛、温暖人心 | 亲情、友情、励志 |

---

## 处理流程（详细）

### 步骤1：场景检测

```bash
python3 detect_scenes.py input.mp4 output/scenes.json --threshold 30
```

### 步骤2：关键帧提取

```bash
python3 extract_keyframes.py input.mp4 output/scenes.json output/keyframes/ --max-frames 5
```

### 步骤3：画面描述

```bash
python3 describe_scenes.py output/keyframes/ output/descriptions.json
```

### 步骤4：生成解说文案

```bash
python3 generate_narrator.py output/descriptions.json output/narrator.txt --style 幽默 --format both
```

## SKILL.md 核心内容

### 触发条件

用户满足以下任一条件时使用此技能：
- 用户粘贴了本地视频文件路径并要求画面解说
- 用户提到"无声视频解说"、"画面分析"、"纯视觉解说"
- 用户提到需要风格化解说（风趣幽默、技术硬核等）

### 视频类型选择

| 类型 | 说明 |
|------|------|
| 无声视频（默认）| 纯画面视频，使用场景检测+关键帧分析 |
| 有声视频 | 有音轨但需要画面分析的视频 |

### 风格选择

```
请选择解说风格（直接回复数字或风格名）：
1. 风趣幽默（默认）- 轻松调侃的语气
2. 技术硬核 - 专业术语、深度分析
3. 理性科普 - 客观陈述、逻辑清晰
4. 解压治愈 - 柔和舒缓、放松心情
5. 温馨感人 - 情感充沛、温暖人心
```
